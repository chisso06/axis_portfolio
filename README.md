# コミュニティ内ポートフォリオサイト

## 目的
コミュニティ内で誰がどんな挑戦をしているかを記録する場所を作るため。

## 開発言語
Ruby on Rails

## 制作期間
2ヶ月

## 実装した機能
- ユーザに関わる機能
  - 新規登録
  - ログイン・ログアウト
  - 一覧ページ
  - プロフィールページ
  - 設定
- 投稿に関わる機能
  - 新規投稿
  - 投稿編集
  - 投稿詳細ページ
- その他
  - トップページ
  - ヘルプページ
